# 🔍 파일 무결성 검사기 (File Integrity Checker)

SHA-256 해시를 이용해 파일의 변조, 삭제, 추가 여부를 탐지하는 도구.

---

## 개요

파일이 몰래 수정되거나 삭제되었는지 탐지하는 보안 도구임.
디지털 포렌식에서 증거물의 무결성을 검증하는 원리를 Python으로 구현함.

---

## 주요 기능

- SHA-256 해시 기반 파일 지문 생성
- 스냅샷(기준선) 저장 → `snapshot.json`
- 변조·삭제·추가된 파일 자동 탐지
- 검사 이력 로그 자동 기록 → `integrity_log.txt`
- 존재하지 않는 경로, 손상된 스냅샷 등 예외 상황 안내
- 검증 대상 폴더가 스냅샷 생성 당시 폴더와 다를 경우 경고 표시

---

## 사용 방법

### 1. 스냅샷 생성 (기준선 설정)
```bash
python3 checker.py --snapshot ./검사할폴더
```

### 2. 무결성 검증 (변조 탐지)
```bash
python3 checker.py --verify ./검사할폴더
```

`--snapshot`과 `--verify`는 동시에 지정할 수 없음.

---

## 실행 결과 예시

```
==================================================
    📋 무결성 검사 결과 보고서
==================================================
  스냅샷 기준 시각: 2025-01-01T10:00:00
  검사 완료 시각:   2025-01-01T10:05:00
  검사 파일 수:     2개
--------------------------------------------------

⚠️  변조된 파일 (1개):
   • doc1.txt

✅ 삭제된 파일: 없음
✅ 새로 추가된 파일: 없음
==================================================
🚨 총 1건의 변경이 감지되었습니다.
```

---

## 유의사항

- 스냅샷 파일이 없는 상태에서 `--verify`를 실행하면 안내 메시지 출력 후 종료함
- 존재하지 않는 폴더 경로를 입력하면 오류를 안내함
- `snapshot.json`이 손상된 경우 재생성을 안내함

## 배운 점 / 활용 분야

- 디지털 포렌식: 증거물 수집 시 원본 무결성 증명에 해시값을 사용함
- 보안 모니터링: SIEM, EDR 등 실무 보안 솔루션의 핵심 원리에 해당함
- 사이버 수사: 파일 변조 여부를 객관적 수치로 입증할 수 있음
