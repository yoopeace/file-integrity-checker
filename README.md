# 🔍 파일 무결성 검사기 (File Integrity Checker)

SHA-256 해시를 활용해 파일의 변조, 삭제, 추가를 자동으로 탐지하는 디지털 포렌식 도구입니다.

---

## 📌 프로젝트 소개

파일이 **몰래 수정되거나 삭제되었는지** 탐지하는 보안 도구입니다.  
디지털 포렌식에서 증거물의 무결성을 검증하는 원리를 Python으로 직접 구현했습니다.

---

## ⚙️ 주요 기능

- ✅ SHA-256 해시 기반 파일 지문 생성
- ✅ 스냅샷(기준선) 저장 → `snapshot.json`
- ✅ 변조 / 삭제 / 추가된 파일 자동 탐지
- ✅ 타임스탬프 포함 로그 자동 기록 → `integrity_log.txt`

---

## 🛠 사용 기술

- Python 3
- hashlib (SHA-256)
- json, os, logging, argparse

---

## 🚀 사용 방법

### 1. 스냅샷 생성 (기준선 만들기)
```bash
python3 checker.py --snapshot ./검사할폴더
```

### 2. 무결성 검증 (변조 탐지)
```bash
python3 checker.py --verify ./검사할폴더
```

## 📋 실행 결과 예시

```
==================================================

  검사 파일 수:     2개
--------------------------------------------------

⚠️  변조된 파일 (1개):
   • doc1.txt

✅ 삭제된 파일: 없음
✅ 새로 추가된 파일: 없음
==================================================
🚨 총 1건의 변경이 감지되었습니다.
```

## 💡 배운 점 / 활용 분야

- **디지털 포렌식**: 증거물 수집 시 원본 무결성 증명에 해시값 사용
- **보안 모니터링**: SIEM, EDR 등 실무 보안 솔루션의 핵심 원리
- **사이버 수사**: 파일 변조 여부를 객관적 수치로 입증 가능
